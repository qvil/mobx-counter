# [Mobx Counter](https://github.com/qvil/mobx-counter)
Simple Mobx Counter

## [Demo](https://qvil.github.io/mobx-counter/)

![demo.gif](demo.gif)

## Prerequisites
- Node.js >= 8.x (Recommended the latest version)

## Install
```
npm install
```

## Start
```
npm start
```
